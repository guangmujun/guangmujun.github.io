基于Hugo和Github搭建个人博客

Written by Yuhang.W with ♥ on July 7, 2020 in 博客搭建

步骤

  1. 下载hugo，添加到环境变量

  2. 测试hugo是否安装成功hugo version

  3. 下载hugo主题，修改主题中的配置文件config.toml

  4. 本地启动hugo，查看效果hugo server

  5. 新建一篇博文hugo new posts/blog.md，头部的draft参数设置成False

  6. 写作，本地启动查看博文效果

  7. 生成网页，hugo，生成public目录

  8. Github新建仓库，仓库名为username.github.io

  9. 将public目录下所有文件上传到仓库中

    git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/username/username.github.io  # 使用自己的用户名
git push -u origin master  # 然后输入自己的用户名和密码
git push

  10. 访问https://username.github.io，搭建成功

常见问题

Q1：博文不显示？

A1：文档头部的draft参数设置为False即可。
